# PRD — 股票 LINE Bot

## 目標

讓使用者在 LINE 內透過簡短指令查詢台股即時報價、取得 AI 投資面分析、維護自選股清單。

## 使用者

- **誰**:對台股有興趣的散戶或學生
- **情境**:不想開券商 App 或網頁,只想在通訊軟體裡快速問股價或聽 AI 怎麼看

## 功能清單

| 使用者輸入 | 系統回應 |
|---|---|
| `查詢 2330` | 回傳該股當下的即時開盤/最高/最低/成交價 |
| `分析 2330` 或 `分析 台積電` | 呼叫 Gemini,回傳基本面、產業地位、近期題材、風險的簡短分析 |
| `追蹤 2330` | 將 2330 加入該使用者的 watchlist;若已存在則告知 |
| `清單` | 列出該使用者目前追蹤的所有股號 |
| `刪除 2330` | 從該使用者 watchlist 移除 2330 |
| 其他文字(含「選單」「help」) | 回傳指令說明 |

每筆對話(訊息 + 回覆)會寫進 `interactions` 表,首次互動的使用者會被寫進 `users` 表。

## 非功能需求

- **回應時間**:Webhook 必須在 30 秒內回 200(LINE 限制),Gemini 推論預期 < 5 秒
- **資料保存**:使用者 ID 與互動紀錄存在本機 SQLite,不上雲
- **安全**:三個 API token 透過 `.env` 注入,不寫死在程式碼;LINE webhook 簽章必驗證
- **錯誤容忍**:外部服務(twstock / Gemini)失敗時回友善訊息,不讓 webhook 500

## 範圍外(明確不做)

- 不做下單 / 交易功能
- 不做美股、加密貨幣
- 不做圖表、K 線
- 不做主動推播(沒有 push 通知)
- 不做使用者註冊或登入(用 LINE userId 直接認人)
- 不做多使用者共享 watchlist
- 不做歷史價格、技術指標
